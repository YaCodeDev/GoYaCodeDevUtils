# Security Policy

## Supported Versions

We do only support latest version of library.

| Version  | Supported          |
| -------- | ------------------ |
| latest   | :white_check_mark: |
| < latest | :x:                |

## Reporting a Vulnerability

You can report a vulnerability by sending an email to security@yacode.dev.
While reporting, make sure to include a description on how to reproduce the issue. Make sure to include #GoYaCodeDevUtils in your email so it doesn't get lost.
